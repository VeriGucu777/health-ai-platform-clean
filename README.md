# Health AI Platform

AI-powered health risk analysis platform developed with Flask and machine learning.

## Project Overview

This project is a health-focused AI platform that brings together:

- Heart disease risk analysis
- Diabetes risk analysis
- PDF report generation
- Data visualization
- User-friendly web interface

The platform allows users to enter health-related values, receive a prediction result, and download a PDF report.

## Features

- Heart disease risk prediction
- Diabetes risk prediction
- Flask-based web application
- PDF report download
- Data visualization support
- Simple and structured interface

## Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- ReportLab
- HTML / CSS

## Project Structure

- `app.py` → main Flask application
- `templates/` → HTML pages
- `static/` → visual files
- `model.pkl` → machine learning model
- `diabetes.csv` → dataset
- `heart/` and `health_platform/` → project-related folders

## Purpose

The goal of this project is to build a practical AI-based health risk analysis platform and present prediction results in a clear and useful format.
